# Hospital Management Dashboard

## 1. Project Overview
This project analyzes hospital data to understand patient flow,
doctor performance, hospital facilities, and financial metrics.

## 2. Tools Used
- Power BI
- SQL

## 3. Dataset Description
The dataset includes patient details, doctor information,
hospital facilities, and billing data.

## 4. Data Cleaning (SQL)
- Removed duplicate records
- Handled NULL values using COALESCE
- Standardized date formats
- Joined multiple tables for analysis

## 5. Dashboard Pages
1. Overview – hospital KPIs and summary
2. Patient – patient count, age group, gender analysis
3. Doctor – doctor performance and specialization
4. Hospital Facility – beds, rooms, availability
5. Finance – revenue, expenses, profit analysis

## 6. Key Insights
- Peak patient inflow observed on weekdays
- Certain departments generate higher revenue
- Doctor workload varies significantly

## 7. Conclusion
This dashboard helps hospital management in
better decision-making and resource planning.
